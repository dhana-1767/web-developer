🚀 GitHub Actions CI/CD Workflow

This workflow automatically builds, tests, and deploys your application whenever code is pushed to GitHub.

Workflow File

name: CI/CD Pipeline

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-node@v4
        with:
          node-version: 20

      - run: npm install
      - run: npm test
      - run: npm run build

Features

  • ✅ Automated Build
  • ✅ Automated Testing
  • ✅ Continuous Integration
  • ✅ Continuous Deployment
  • ✅ Artifact Management
